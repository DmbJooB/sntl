# Sunu Nataal Web

A modern web application for the "Sunu Nataal" photographic collective in Senegal.

## Features
- **Image Bank**: Browse and purchase high-quality photographs.
- **Walks & Galleries**: Discover photographic walks and events across Senegal.
- **Admin Dashboard**: Comprehensive management for photographers, images, and events.
- **Shopping Cart & Favorites**: Save and purchase your favorite shots.
- **Watermarking**: Automated subtle watermarking for image protection.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS (Vanilla CSS for custom styles).
- **Backend**: Firebase (Firestore, Authentication, Storage).
- **UI Icons**: Lucide React.
- **Routing**: React Router DOM.

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Start the development server: `npm run dev`.

## Deployment
The project is configured for Firebase Hosting. Use `firebase deploy` to publish changes.
